# E97: Earth Core Seismic Speed from Lattice Compression

> **Category**: Gravity / Occlusion
> **Difficulty**: LOW
> **Status**: Archival analysis

---

## 1. Background

SDT predicts seismic velocities partially reflect lattice compression, which increases toward Earth's centre.

## 2. SDT Prediction

Seismic wave speeds in Earth's core should correlate with the convergence-field compression gradient.

## 3. Standard Prediction

Standard: seismic speeds from material properties (density, bulk/shear moduli).

## 4. Observable

Compare seismic travel times to SDT lattice compression model.

## 5. Apparatus

Global seismograph network data.

## 6. Feasibility

LOW — archival.

## 7. Impact

- **If confirmed**: If correlation exists: supports lattice compression gradient.
- **If not confirmed**: If no correlation: seismic speeds are purely material properties.
