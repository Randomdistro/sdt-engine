# E73: Local Speed of Light Measurement Near Dense Object

> **Category**: Electromagnetic
> **Difficulty**: LOW
> **Status**: Archival analysis

---

## 1. Background

SDT predicts the lattice relay speed changes near dense objects because the lattice is compressed.

## 2. SDT Prediction

The local speed of light near a very dense object (neutron star, white dwarf) should be measurably different from c.

## 3. Standard Prediction

Standard: c is constant; what changes is the coordinate speed (time dilation).

## 4. Observable

Measure timing of pulsar emission through different density environments.

## 5. Apparatus

Pulsar timing data.

## 6. Feasibility

LOW — archival.

## 7. Impact

- **If confirmed**: Timing anomalies near dense objects would support variable relay speed.
- **If not confirmed**: If timing is clean: lattice compression doesn't measurably affect c.
