### The EC Mechanism

When a nucleus has an He-3 core, the bare proton at 1s2 reels in a valence proton via electron capture:

1. Core vacancy: 1s2 needs a neutron
2. Core reels in a proton from the next shell (from a valence deuteron)
3. Proton + 1s electron → neutron (the EC event) + neutrino receipt
4. New neutron fills 1s2 vacancy → core completes as He-4
5. Donor deuteron lost its proton → orphan neutron remains
6. Orphan neutron joins adjacent deuteron → triton
7. Result: **He-4 core + restructured valence. Grammar resolved.**
