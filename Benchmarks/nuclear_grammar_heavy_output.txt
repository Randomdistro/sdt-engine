── PART 3: POLONIUM-214 — THE MIDPOINT ANALYSIS ──

  U-238:  alpha + 36d + 54t
  Po-214: alpha + 36d + 46t
  Pb-206: alpha + 38d + 42t

  U-238 -> Po-214: lost 8t + 0d
  Po-214 -> Pb-206: lost 4t + -2d

  KEY INSIGHT:
    U-238 to Po-214: exactly 8 triton bonds removed
    That's the number James predicted: 8 triton bonds on each side
    The alpha core + deuteron scaffold is PRESERVED
    Only tritons are shed (as alpha particles + beta adjustments)
