# Open Problems

Three computational problems remain. Zero conceptual gaps.

## 1. R/a Quantisation

**Problem**: Which specific aspect ratios of the genus-1 torus produce self-consistent mode-locked equilibria?

**What we know**: The modal coupling coefficient $\mathcal{C}_{TP} = a/R$ controls the coupling strength between toroidal and poloidal modes. The equilibrium mode velocities ($v_T, v_P, v_C$) are completely determined by $R/a$ and the helicity angle $\alpha$.

**What we need**: A lattice simulation that solves the coupled mode equations and finds the discrete set of $(R/a, \alpha)$ values where all three modes self-consistently balance.

**What it predicts**: The discrete mass spectrum — which specific exclusion volumes (and therefore masses) are stable.

## 2. Winding Number Quantisation

**Problem**: Why are $W=1$ (electron) and $W=3$ (proton) the only stable winding numbers?

**What we know**: Higher winding amplifies centrifugal pressure per mode (unlike higher genus which dilutes it). $W=3$ is a trefoil — the simplest non-trivial knot. $W=2$ would be a torus link, not a knot.

**What we need**: Lattice simulation of vortex reconnection dynamics. Specifically: does a $W=2$ configuration spontaneously reconnect to $W=1$ + energy, while $W=3$ (being knotted) cannot?

**What it predicts**: The complete stable particle catalogue from topology alone.

## 3. Fine Structure Constant from Topology

**Problem**: Can $\alpha = 1/137.036$ be derived from the W=1 torus geometry?

**Conjecture**: $\alpha = v_P/v_T$ at the W=1 equilibrium — the ratio of poloidal to toroidal circulation speed. This would make $\alpha$ a geometric property of the simplest stable torus.

**What it predicts**: If correct, this eliminates $\alpha$ as an independent input. The terminal input set becomes: $\ell_P$, $T_{\text{CMB}}$, $d=3$.
